# __Decreased droplet transmission__

## Stakeholders benefitting from this Benefit

* [workers](https://github.com/mm80843/T3.5/blob/main/docs/Stakeholder/PBN__Stakeholder_128.md)
* [employers](https://github.com/mm80843/T3.5/blob/main/docs/Stakeholder/PBN__Stakeholder_180.md)

## Name of the Benefit

Physical barriers, such as plexiglass shields, can create a physical barrier between workers, reducing the risk of droplet transmission of the virus.

## Description of Benefits pertaining to Beneficiaries

* [Implementing physical barriers can create a safer work environment, reducing the risk of COVID-19 transmission among workers and potentially decreasing the number of sick days taken. This can contribute to improved employee health, increased productivity, and potentially reduced healthcare costs for employers.](https://github.com/mm80843/T3.5/blob/main/docs/BenefReturn/PBN__BenefReturn_399.md)
