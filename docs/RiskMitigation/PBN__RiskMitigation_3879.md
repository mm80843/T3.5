# __Unbiased data collection__

## Property: has_MitigationPrinciple

Collecting data using robust methods and unbiased sampling techniques to avoid biased estimates and inferences about contagious diseases and their impact, prevalence rates, transmission, and effectiveness of mitigation measures.

## Risks possibly mitigated by this Mitigation

* [Biased estimates and inferences](https://github.com/mm80843/T3.5/blob/main/docs/Risk/PBN__Risk_2755.md)
